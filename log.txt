# imarisfront
this is imari+, a system to help small business track daily expenses and calculate payroll as well as save for it periodically.

## What's done so far

Created the fronted in React + TypeScript
and the backend in Node.js + Express + TypeScript

So far
I've created the physicians, and Transactions APIs
also just finished doing (07/01/26) the Users API, there are still thing missing, ie, login, password hashing, jwt, validation and auditing which will be done in due time
